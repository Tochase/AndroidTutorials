# 2.3  Activity Intent

显式 Intent 启动 SecondActivity，隐式 Intent 启动 ThirdActivity。

隐式 Intent 打开链接等。

FourthActivity 响应打开链接操作。

通过 Intent，MainActivity 向 SecondActivity 传递数据。

通过 Intent，SecondActivity 向 MainActivity 反向传递数据。
